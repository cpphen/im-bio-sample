# im-bio-sample

## Steps to start project

- Node version used 24.4.0
- cd into imBioReactNative
- run command `npm install`
- Open Android Studio and open project folder
- Start an android emulator
- Once emulator is started, in a separate terminal run command `npx react-native run-android`
- If successful, application should be visible in the Android emulator

- As an alternative method, in separate terminal windows, first run the command `npx react-native start`
- Once server is started with `npx react-native start` run command `npx react-native run-android` in a separate terminal
